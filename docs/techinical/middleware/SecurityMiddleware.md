## üîí Headers de Seguran√ßa

- **Prop√≥sito**: Ativa prote√ß√£o XSS do navegador

- **Prop√≥sito**: Impede MIME type sniffing

- **Configur√°vel**: `referrerPolicy => 'valor'`
